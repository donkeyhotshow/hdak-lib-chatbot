# Улучшения доступности

## Критические проблемы

### 1. Контрастность цветов
- Золотой текст на светлом фоне: недостаточный контраст
- Серый текст в сайдбаре: плохо читается

### 2. Навигация с клавиатуры
- Отсутствует skip-to-content ссылка
- Неполная поддержка Tab-навигации
- Нет индикаторов фокуса на всех элементах

### 3. Семантическая разметка
- Отсутствуют landmarks (main, nav, aside)
- Неправильное использование heading hierarchy
- Нет aria-labels для интерактивных элементов

## Решения

### Улучшение контрастности
```css
:root {
  --text-high-contrast: #2d1f14; /* Вместо золотого */
  --sidebar-text: #f5f5f5; /* Вместо полупрозрачного */
}
```

### Семантическая структура
```html
<main role="main" aria-label="Чат с библиотечным помощником">
  <nav role="navigation" aria-label="Главное меню">
  <aside role="complementary" aria-label="Быстрые действия">
```

### Клавиатурная навигация
- Добавить skip-links
- Улучшить focus indicators
- Поддержка Escape для закрытия модалов
