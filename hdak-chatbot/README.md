## Що зроблено

### Сайдбар (Sidebar.tsx)
- Логотип: SVG книга + "Бібліотека / ХДАК / ЧАТ-ПОМІЧНИК БІБЛІОТЕКИ"
- Секція ШВИДКІ ЗАПИТИ — 4 кнопки що запускають чат
- Секція РЕСУРСИ БІБЛІОТЕКИ — 4 посилання з підписами
- Кнопка "Більше ресурсів" — розгортає 4 додаткові посилання
- Секція АРХІВ СЕСІЙ — показує 3 останні чати, решта ховається
- Кнопка НОВИЙ ЧАТ — скидає розмову, повертає на лендинг
- Overlay backdrop при відкритті
- Slide-in анімація

### Лендинг (page.tsx)
- 4 картки: Графік / Записатися / Каталог / Контакти
- Горизонтальний layout: іконка + назва + підпис
- Hover: підйом + золота рамка
- Поле вводу під картками
- Заголовок: "Бібліотека ХДАК / Чим можу допомогти?"

### Чат (page.tsx)
- Швидкі відповіді для 12 тем (без затримки AI)
- Streaming для інших запитів
- Збереження в localStorage
- Typing indicator (3 крапки)
- Кнопки: копіювати / 👍 / 👎
- Чипи-підказки після кожної відповіді
- Роздільник "сьогодні"

### Автовідповіді (hdak-data.ts)
12 тем з реальними даними lib-hdak.in.ua:
1. Графік роботи (3 відділи з точним розкладом)
2. Запис до бібліотеки (особисто + дистанційно)
3. Електронний каталог (посилання + поради)
4. Контакти та адреса (всі канали зв'язку)
5. Правила бібліотеки (7 пунктів)
6. Е-читальна зала (графік, послуги, заборони)
7. Репозитарій ХДАК
8. Наукові бази даних (відкриті + корпоративні)
9. Авторські профілі (ORCID, Scopus)
10. Єдина картка читача
11. Структура бібліотеки (директор, відділи)
12. Нові надходження
